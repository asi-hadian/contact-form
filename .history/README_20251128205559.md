## Overview

This project is a modern, responsive contact form built with HTML, CSS, and JavaScript. The form allows users to submit their information, select a query type, and provides real-time validation feedback for a seamless user experience. Custom radio buttons and styled form inputs enhance the UI, and a success message is displayed upon submission.

## Features

- **Responsive Design:** Works on desktop and mobile devices; form fields stack vertically on smaller screens.
- **Client-Side Validation:** Required fields, email format validation, and checkbox consent verification.
- **Custom Radio Buttons:** Radio buttons display custom images when selected and highlight the parent container.
- **Accessible Form:** ARIA attributes, `aria-live` regions, and keyboard navigation support.
- **Error Highlighting:** Invalid fields are highlighted with a red border.
- **Success Message:** Displays a confirmation message when the form is submitted successfully.
- **Clear Form After Submission:** All form inputs are cleared after successful submission.
